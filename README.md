# Foyer Management Backend - Foyer2324

## Description
A Spring Boot backend for managing university housing (foyer). Handles student data, room allocation, and housing management with secure REST APIs. Developed as part of a fourth-year engineering project.

## Features
- Manage students and rooms
- Allocate and track housing
- Secure REST API endpoints
- Data persistence with MySQL or preferred DB

## Technologies Used
- Java 17 / Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Setup & Installation
1. Clone the repository
2. Configure `application.properties` with your DB settings
3. Build with Maven:
```bash
mvn clean install
4. Run the application:
mvn spring-boot:run
